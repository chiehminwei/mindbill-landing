# MindBill — Landing Page

AI receptionist + revenue cycle for psychiatric, TMS, and Spravato practices.

**Live**: [mindbill.org](https://mindbill.org)

## Stack
- Single-file HTML + Tailwind CDN (no build step)
- Hosted on Vercel
- Auto-deploys on push to `main`

## Local preview
```bash
open index.html
```

## Deploy
Pushes to `main` deploy automatically via Vercel's GitHub integration.
